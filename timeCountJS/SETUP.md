# Быстрая установка бэкенда

## Шаг 1: Создание базы данных

1. Откройте phpMyAdmin: http://localhost/phpmyadmin
2. Создайте новую базу данных `timecount_db` или импортируйте файл `api/database.sql`

## Шаг 2: Настройка подключения

Откройте `api/config.php` и при необходимости измените параметры:
```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'timecount_db');
define('DB_USER', 'root');      // Ваш пользователь MySQL
define('DB_PASS', '');          // Ваш пароль MySQL
```

## Шаг 3: Проверка работы

1. Откройте в браузере: `http://localhost/timeCountJS/timeCountJS/api/get_statistics.php`
2. Должен вернуться JSON ответ с пустой статистикой

## Готово! 

Теперь при использовании приложения:
- Сессии автоматически сохраняются в базу данных
- При нажатии "Данные за день" статистика загружается с сервера
- Если сервер недоступен, данные сохраняются локально и синхронизируются позже
